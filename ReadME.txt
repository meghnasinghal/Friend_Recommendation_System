To run the jars, the commands are as follows:

Q1: hadoop jar AssignmentQ1.jar <input path> <output path>
eg: hadoop jar CommonFriendList.jar CommonFriendList /input/soc-LiveJournal1Adj.txt /outputQ1

Q2: hadoop jar AssignmentQ2.jar <input path> <output path 1> <output path 2>
eg: hadoop jar MaximumMutualFriends.jar MaximumMutualFriends /input/soc-LiveJournal1Adj.txt /outputp1 /outputQ2

Q3: hadoop jar AssignmentQ3.jar ClassName <input path 1>  <input path 2>  <friend 1 id> <friend 2 id> <output path>
eg: hadoop jar UserInfo.jar UserInfo /input/soc-LiveJournal1Adj.txt /input/userdata.txt 28041 28056 /outputQ3

Q4: hadoop jar AssignmentQ4.jar ClassName <input path 1>  <input path 2>  <output path > 
eg: hadoop jar FriendsMaxAge.jar FriendsMaxAge /input/soc-LiveJournal1Adj.txt /input/userdata.txt /outputQ4
